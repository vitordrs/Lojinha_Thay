## 📖 Sobre o projeto

A **Lojinha da Thay** é uma página simples de apresentação de produtos, simulando uma pequena loja virtual.

O projeto foi desenvolvido com foco em **estruturação de layout e responsividade**, utilizando apenas **HTML e CSS**, sem uso de JavaScript ou frameworks.

A página possui:

* Seção **Sobre a loja**
* Área de **produtos**
* **Formulário de contato**
* Layout adaptado para **mobile, tablet e desktop**

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Media Queries (responsividade)

---

## 📱 Responsividade

O layout foi adaptado para diferentes tamanhos de tela:

* 📱 **Mobile**
* 💻 **Desktop**
* 📲 **Tablet**

Utilizando **Media Queries** para reorganizar os elementos e melhorar a experiência do usuário em cada dispositivo.

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

* Estruturação semântica com **HTML**
* Estilização com **CSS**
* **Layouts responsivos**
* Uso de **Grid e Flexbox**
* Organização de seções em uma página web
